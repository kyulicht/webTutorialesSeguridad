Kali Linux, Virtualización y Protocolos de Red

Kali Linux

Kali Linux es una distribución de Linux basada en Debian diseñada para pruebas de penetración y auditorías de seguridad. Viene pre-instalada con numerosas herramientas de seguridad y hacking.

Virtualización con VirtualBox

VirtualBox es un software de virtualización que permite ejecutar múltiples sistemas operativos en una sola máquina física. Es útil para:

  • Probar diferentes sistemas operativos sin necesidad de instalarlos directamente en el hardware.
  • Crear entornos aislados para pruebas de software o seguridad.
  • Ejecutar aplicaciones que requieren un sistema operativo específico.

Para virtualizar Kali Linux con VirtualBox:

  1. Descarga e instala VirtualBox desde el sitio oficial.
  2. Descarga la imagen ISO de Kali Linux.
  3. En VirtualBox, crea una nueva máquina virtual y asigna recursos (RAM, CPU, espacio en disco).
  4. Configura la máquina virtual para arrancar desde la ISO de Kali Linux.
  5. Inicia la máquina virtual e instala Kali Linux.

10 Comandos Comunes en Kali Linux

  1. nmap: Escáner de puertos y auditoría de seguridad de red.
  2. metasploit: Framework para desarrollo y ejecución de exploits.
  3. aircrack-ng: Suite para auditar redes inalámbricas.
  4. hydra: Herramienta para crackeo de contraseñas en red.
  5. john: John the Ripper, crackeador de contraseñas offline.
  6. wireshark: Analizador de protocolos de red.
  7. sqlmap: Herramienta para detectar y explotar vulnerabilidades SQL injection.
  8. nikto: Escáner de vulnerabilidades web.
  9. hashcat: Herramienta avanzada para recuperación de contraseñas.
  10. burpsuite: Plataforma integrada para realizar pruebas de seguridad en aplicaciones web.

FTP (File Transfer Protocol)

FTP es un protocolo de red para la transferencia de archivos entre sistemas conectados a una red TCP, basado en una arquitectura cliente-servidor.

8 Protocolos de Red Comunes

  1. HTTP/HTTPS: Protocolo de transferencia de hipertexto, base de la comunicación en la World Wide Web.
  2. TCP/IP: Conjunto de protocolos fundamentales para la comunicación en Internet.
  3. UDP: Protocolo de datagramas de usuario, usado para transmisiones rápidas sin conexión.
  4. DNS: Sistema de nombres de dominio, traduce nombres de dominio a direcciones IP.
  5. SMTP: Protocolo simple de transferencia de correo, usado para el envío de emails.
  6. SSH: Secure Shell, protocolo para acceso remoto seguro a sistemas.
  7. DHCP: Protocolo de configuración dinámica de host, asigna automáticamente direcciones IP.
  8. SNMP: Protocolo simple de administración de red, usado para monitorear y gestionar dispositivos en red.
