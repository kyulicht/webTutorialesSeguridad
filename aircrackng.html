Tutorial de Hacking Ético: Aircrack-ng y Airodump-ng

Introducción a Aircrack-ng

Aircrack-ng es una suite de herramientas usadas para auditar la seguridad de redes Wi-Fi. La herramienta principal, aircrack-ng, se utiliza para descifrar contraseñas WEP y WPA/WPA2 capturando paquetes de datos.

Comandos Básicos de Aircrack-ng

La estructura básica del comando es:

aircrack-ng -a [modo] -b [BSSID] -w [ruta/diccionario.txt] [captura.cap]

Algunos parámetros importantes:

  • -a: Especifica el modo de ataque (1 para WEP, 2 para WPA/WPA2).
  • -b: Especifica la dirección MAC del punto de acceso (BSSID).
  • -w: Especifica la ruta del diccionario de contraseñas.

Ejemplo básico:

aircrack-ng -a 2 -b 00:11:22:33:44:55 -w /path/to/dictionary.txt captura.cap

En este ejemplo, Aircrack-ng intenta descifrar la contraseña WPA/WPA2 usando un diccionario de contraseñas.

Introducción a Airodump-ng

Airodump-ng es una herramienta dentro de la suite Aircrack-ng que se utiliza para capturar tráfico de red inalámbrica. Es útil para recolectar datos que luego pueden ser usados en ataques de fuerza bruta con Aircrack-ng.

Comandos Básicos de Airodump-ng

La estructura básica del comando es:

airodump-ng [opciones] [interface]

Algunas opciones importantes:

  • --bssid: Fija la captura al BSSID específico del punto de acceso.
  • --channel: Limita la captura a un canal específico.
  • -w: Especifica el nombre de archivo para guardar la captura.

Ejemplo básico:

airodump-ng --bssid 00:11:22:33:44:55 --channel 6 -w captura wlan0

En este ejemplo, Airodump-ng capturará el tráfico del BSSID específico en el canal 6 y lo guardará en un archivo llamado "captura".

Situaciones de Uso

Estas herramientas se usan principalmente en auditorías de seguridad para probar la robustez de contraseñas de redes inalámbricas. Airodump-ng se utiliza primero para capturar paquetes de datos, y luego Aircrack-ng se usa para intentar descifrar la contraseña de la red.
