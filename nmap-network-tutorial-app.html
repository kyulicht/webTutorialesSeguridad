Tutorial: Nmap y Escaneo de Redes

¿Qué es Nmap?

Nmap (Network Mapper) es una herramienta de código abierto utilizada para descubrir redes y auditar la seguridad. Permite a los administradores de sistemas y profesionales de seguridad escanear redes y sistemas para determinar qué puertos están abiertos, qué servicios se están ejecutando y otras características.

Tipos de Redes

  • LAN (Local Area Network): Red local que cubre un área pequeña como una oficina o un hogar.
  • WAN (Wide Area Network): Red que cubre un área geográfica amplia, como una ciudad o país.
  • VPN (Virtual Private Network): Red privada que utiliza una red pública (como Internet) para conectar usuarios remotos.
  • VLAN (Virtual Local Area Network): Red lógica independiente dentro de una red física.

Comandos Básicos de Nmap

Comando Descripción
nmap [IP] Escaneo básico de los 1000 puertos más comunes
nmap -p- [IP] Escanea todos los puertos (1-65535)
nmap -sV [IP] Detecta versiones de servicios
nmap -O [IP] Intenta detectar el sistema operativo
nmap -sU [IP] Escaneo UDP

Qué hacer si no se pueden escanear los puertos

  1. Verificar conectividad: Asegúrate de que tienes conectividad con el objetivo usando ping o traceroute.
  2. Comprobar firewalls: Los firewalls pueden bloquear los escaneos. Intenta usar técnicas de evasión de Nmap.
  3. Usar escaneo sigiloso: Utiliza la opción -sS para un escaneo SYN sigiloso.
  4. Ajustar la velocidad: Un escaneo más lento puede evitar la detección. Usa -T2 para un escaneo más lento.
  5. Fragmentación de paquetes: Usa -f para fragmentar los paquetes y evadir algunos firewalls.

Parámetros Avanzados de Nmap

  • -sN: Escaneo NULL
  • -sF: Escaneo FIN
  • -sX: Escaneo XMAS
  • -Pn: Trata todos los hosts como en línea (omite el descubrimiento)
  • -A: Habilita la detección de SO, versión, script y traceroute
